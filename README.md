## ng2 + tsc + unit tests

Always updated with the latest version of  ng2 (beta-5), tsc (1.8) and jasmine (2.3).

### Install

```shell
$ npm i
```

### Install definitions

```shell
$ typings install jasmine --ambient -so
```

### Compile

```shell
$ npm run build
```

### Run

```shell
$ npm start
```

### Test

```shell
$ npm t
```

## Happy coding!
