## ng2 + tsc + unit tests

Always updated with the latest version of  ng2 (beta-7), tsc (1.8) and jasmine (2.3).

### Install dev dependencies

```shell
$ npm i
```

### Install typings stuff

```shell
$ typings install
```

### Compile

```shell
$ npm run build
```

### Run

```shell
$ npm start
```

### Test

```shell
$ npm t
```

## Happy coding!
