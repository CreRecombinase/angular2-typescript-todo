### NG2 + TSC + Unit Tests

Always updated with the latest version of  ng2 (.50), tsc (1.7) and jasmine (2.3).

### Install all the npm dependencies

```shell
$ npm i
```

### Compile Typescript

```shell
$ npm run build
```

### Run the server

```shell
$ npm start
```

### Run the Tests

```shell
$ npm t
```

## Happy coding!
